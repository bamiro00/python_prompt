# 반려동물 추억 콘텐츠 프롬프트 관리 프로그램

반려동물과 함께한 소중한 순간을 글, 이미지, 영상 콘텐츠로 만들기 위한 프롬프트를 관리하는 Python 콘솔 프로그램이다.

## 기본 프롬프트

- 반려동물 성장일기 작성
- 반려동물 추억 일러스트 기획
- 반려동물 추억 영상 스토리보드
- 반려동물 추억 작가
- 반려동물 사진 정리 계획

## 주요 카테고리

- 텍스트 생성
- 이미지 생성
- 영상 생성
- 페르소나
- 자동화
- 기타

## 현재 구현 상태

- 카테고리 목록 정의
- 기본 프롬프트 5개 등록
- 즐겨찾기 기본값 설정
